Active Ports in Nova Scotia

⬅ Back

This viz uses data taken from the PUB150 World Port Index ↗, a database of all active ports in the world, to map the active ports of Nova Scotia to a digital canvas.

The latitude and longitude coordinates of each port are accurately translated to the canvas using a mapping that adjusts these geographical points to fit within the predefined bounds of the canvas.

Each port is represented by a circle whose size corresponds to the harbor's relative size. The colors of these circles vary based on the channel depth; lighter shades denote shallower areas, while darker blues show deeper waters. The stroke weight of each circle reflects the tidal range, with thicker borders representing greater tidal variations. A hovering tooltip provides the port's name.
